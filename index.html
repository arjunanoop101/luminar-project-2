Recent Projects

Explore some of my recent projects below. For more, visit my GitHub profile.

Tomato – Food Delivery Website

A full-stack MERN-based platform for seamless food ordering and delivery. Features secure user authentication, admin panel for menu & order management, and real-time order tracking. Integrated ImageKit for image uploads and optimized the UI for responsive performance. Tech Stack: MongoDB, Express.js, React, Node.js, ImageKit, JavaScript, HTML, CSS.

Finance Tracker Web Application

An intuitive MERN web app to manage multiple financial accounts (wallet, bank, crypto). Supports income/expense categorization with real-time visual analytics (PieCharts & BarCharts). Includes JWT-based authentication, Excel report export, and responsive design with protected routes. Tech Stack: MongoDB, Express.js, React, Node.js, JWT, Chart.js, JavaScript, HTML, CSS.

MERN Stack Blog Website

A feature-rich blogging platform with secure admin authentication, CDN-optimized image uploads via Cloudinary, and user commenting. Enables admins to create, edit, delete posts and supports category-based filtering. Fully responsive interface built with Tailwind CSS and secured using bcrypt & REST APIs. Tech Stack: MongoDB, Express.js, React, Node.js, JWT, bcrypt, Tailwind CSS, Cloudinary, JavaScript, HTML, CSS.
